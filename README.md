# 🐾 PawMatch

**Because every pet deserves a social life**

Application mobile de mise en relation entre propriétaires de chats et chiens.

## Fonctionnalités
- Swipe de profils d'animaux
- Carte géolocalisée (urbain & rural)
- Module reproduction vérifié
- Communauté par race
- Messagerie intégrée
- Profil complet avec photos et vidéo
- Abonnement Premium

## Stack
- React 18
- Déployé sur Vercel
